# BL4 Third Person Camera

Third-person, over-shoulder, and freecam controls for Borderlands 4 implemented as a Python SDK mod with custom native AOB-scanned hooks.

## Features

- Toggle third-person mode
- Toggle shoulder side for over-shoulder view
- Optional ADS behavior overrides
- Adjustable camera FOV and viewmodel FOV
- Freecam with movement and mouse-look controls

## Requirements

- Borderlands 4
- BLSDK / unrealsdk
- `mods_base >= 1.11`
- `unrealsdk >= 1.3.0`
- `oak2_mod_manager`

## Notes

This mod is client-side only and targets BL4 exclusively.
